# Online-Examination-System

The **Online Examination System** is a cross‑platform, user‑friendly exam management application developed in **Java**. It is designed to conduct intra‑organizational exams and assessments, simplifying the process for administrators, instructors, and candidates. The system supports exam scheduling, secure login, instant results, and detailed report generation.

---

## 📖 Introduction
This application streamlines the examination process by providing:
- Seamless administration and scheduling of exams.
- Real‑time feedback and instant results.
- Secure storage and retrieval of exam data via **MySQL**.
- Detailed analysis and visualization of results for instructors.

The system requires a **MySQL server** hosted on the intranet. Candidates log in with credentials provided by administrators to access and attempt exams.

---

## ✨ Features & Highlights
- **Quick Scheduling**: Efficient exam scheduling for groups/batches.  
- **Instant Results**: Immediate feedback after exams.  
- **Secure Data Storage**: Exam data stored in MySQL.  
- **Individual Feedback**: Personalized performance reports.  
- **Result Analysis**: Detailed exam reports with charts/graphs.  
- **Input Validation**: Ensures integrity and security.  
- **Visualization**: Uses **JFreeCharts** for graphical reports.  

---

## 🔑 Modules
### 1. Candidate Module
- Login with credentials.  
- Appear in exams and practice tests.  
- View previous scores and upcoming exams.  
- Manage profile and credentials.  

### 2. Instructor Module
- Prepare and upload exam questions.  
- Schedule/manage exams.  
- Edit exam details.  
- Analyze results and visualize reports.  
- Manage profile and credentials.  

### 3. Administrator Module
- Full control over system data.  
- Manage instructors, students, and groups.  
- Insert, edit, or delete exam information.  
- Collect and distribute results.
